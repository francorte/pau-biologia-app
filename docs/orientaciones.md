# Directrices y Orientaciones PAU Biología 2025-26

## Estructura del examen

1. **5 preguntas** en total
2. **2 obligatorias** (competenciales)
3. **3 con opciones** (elegir entre A y B)
4. **2 puntos** por pregunta
5. **90 minutos** de duración

## Bloques de contenidos

| Bloque | Nombre | Contenido principal |
|--------|--------|---------------------|
| A | Biomoléculas | Glúcidos, lípidos, proteínas, enzimas, ácidos nucleicos |
| B | Genética molecular | ADN, ARN, replicación, transcripción, traducción, mutaciones |
| C | Biología celular | Célula procariota/eucariota, orgánulos, membranas, ciclo celular |
| D | Metabolismo | Catabolismo, anabolismo, respiración, fotosíntesis |
| E | Biotecnología | PCR, ingeniería genética, CRISPR, OMG, aplicaciones |
| F | Inmunología | Sistema inmunitario, antígenos, anticuerpos, vacunas |

## Criterios de corrección

### Se valora positivamente:
- Conocimiento concreto del contenido
- Claridad y capacidad de síntesis
- Esquemas pertinentes
- Lenguaje científico-biológico correcto
- Coherencia, cohesión y corrección gramatical

### Penalizaciones ortográficas:
- 2 primeras faltas: sin penalización
- A partir de la 3ª: -0,10 puntos cada una
- Máximo: -1 punto total
- Errores de redacción/presentación: hasta -0,5 puntos

## Formato de respuesta recomendado

1. **Qué me piden** (1 línea)
2. **Respuesta directa** (idea clave)
3. **Explicación paso a paso** (2-8 líneas)
4. **Esquema** (si aporta)
5. **Cierre** (conclusión breve)
